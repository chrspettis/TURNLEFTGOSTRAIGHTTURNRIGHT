# Turn-Left Go-Straight Turn-Right

An interactive ASL practice game designed to engage students with directional language, map-based choices, and fingerspelling.

Created by Christy Pettis, 2026

## Purpose

This game was designed to be usable both online (via GitHub Pages or LMS) and offline (via direct HTML launch).

## Files

- `index.html`: Splash page and local entry point
- `04_TLGS_Game.html`: Full self-contained game with built-in splash
- `demo_game.html`: Optional simplified game view (if included)
- `03_TLGSTR_StudentInstructions.pdf`: Printable or viewable student directions
- `01_Printable_Story_Scroll_v2m.html`: Fillable story-writing scroll
- `07_SplashPage.png`: Large splash art for intro
- `banner.png`: Optional horizontal banner
- `08_PLI LOGO.png`: Footer logo image
- `README_START_HERE.txt`: Instructions for teachers

## Use

1. Unzip the folder.
2. Open `index.html` in a browser to launch the game.
3. Or upload all files to Canvas and update file IDs inside `04_TLGS_Game.html`.
